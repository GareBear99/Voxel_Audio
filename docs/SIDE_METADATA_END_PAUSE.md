# Side Metadata End Pause — v1.8.7

This release extends the pause at the end of the small side metadata scroll.

## Changed

- The small `VOXEL AUDIO EXPORT • song • artist/feat` line still scrolls right-to-left.
- Once it reaches the end, it now pauses longer before resetting back to the beginning.
- End pause increased from about 1.4 seconds to about 3.6 seconds.

## Explicitly unchanged

- Main large title rendering is untouched.
- Start pause is unchanged.
- Scroll speed is unchanged.
- Only the post-scroll/end hold was adjusted.
