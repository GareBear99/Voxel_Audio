# Export Layering Policy

Export composition is ordered so accent shapes never cover the cover art or panels.

## Layer order

1. Blurred background cover.
2. Dark readability overlay.
3. Background-only RGB accent shapes.
4. Cover image.
5. Title and metadata.
6. Description panel.
7. Reactive circle visualizer.
8. Waveform panel.
9. Watermark / branding.

Accent shapes are intentionally drawn before the cover and panels.
