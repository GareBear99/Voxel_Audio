# Watermark Policy

The local/free browser Record path includes a watermark scaffold.

## Free account plan

- Browser Record exports.
- Watermark enabled.
- 5 exports/week once accounts are released.

## Paid plans

### $3 USD/month

- 99 server FFmpeg exports/month.
- Server-side FFmpeg rendering.

### $7 USD/month

- Unlimited no-watermark browser Record exports.
- Unlimited no-watermark server FFmpeg exports.

## Current local build

The no-server local build includes visible watermark logic in the browser export renderer as product-policy scaffolding. Account checks will disable it for paid tiers later.
