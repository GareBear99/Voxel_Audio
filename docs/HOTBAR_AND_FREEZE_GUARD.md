# Hotbar and Freeze Guard

## Topbar cleanup

Actions duplicated in the Quick Dock are hidden from the top scroll bar:

- Open Folder
- Bind Image
- Info
- Preview
- Record
- MP4 Kit
- Album

The fixed Quick Dock remains the main control strip for core actions.

## Freeze guard

The app reports runtime errors and missing controls through the startup diagnostic panel.

This helps detect local browser issues, missing APIs, or broken optional buttons without making the page look silently frozen.
