# Admin Preview Unlock and Single Export Color

## Admin watermark unlock

The export panel includes:

- Admin name
- Admin key
- Unlock No-Watermark
- Lock Watermark
- status hint

Admin key:

```text
VOXEL-AUDIO-ADMIN
```

When unlocked:

- Live Preview hides the watermark.
- Browser Record hides the watermark.
- MP4 Kit manifest records admin clean-export status.

## Crosscheck

Preflight/Crosscheck reports:

- selected audio
- linked image
- watermark/clean-export status
- active export color

## Single export color

The Export color picker overrides the selected voxel color for the current single-track export.

The default still loads from the selected song voxel color.
