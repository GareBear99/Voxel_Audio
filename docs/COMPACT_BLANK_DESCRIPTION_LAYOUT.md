# Compact Blank Description Layout

When Description is blank, Voxel Audio no longer leaves an empty description area.

## Behavior

- Blank Description uses a compact visualizer capsule.
- The metadata panel height is reduced.
- The reactive circle becomes slightly larger.
- The export layout stays balanced instead of looking empty.

## Description field behavior

- Blank = no description text.
- Typed text = description panel with the typed text.
