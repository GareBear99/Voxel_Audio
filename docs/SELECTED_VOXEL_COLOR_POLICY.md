# Selected Voxel Color Policy

All export visuals now follow the selected song voxel color.

## Applies to

- app accent variables
- preview panel border
- export panel border
- title glow
- metadata text
- description/capsule panel fill and border
- cover outline
- reactive frequency circle
- waveform line
- waveform bars
- progress bar
- watermark box/text
- background glow

## Intent

The selected song voxel color is the visual identity for that song's export.
