# Artist Required Policy

Voxel Audio requires an Artist name before Preview, Record, or MP4 Kit export.

## Rules

- Artist name is required.
- Maximum length: 36 characters.
- Artist name is locked into metadata when the export starts.
- Re-export from Vault also requires an Artist name.

## Why

Artist identity is part of the export metadata and release identity. The cap prevents layout overlap and keeps the metadata row clean.
