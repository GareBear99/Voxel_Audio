# Pricing / Accounts Plan

Planned account tiers once Google OAuth and Stripe are integrated. All prices are **USD**.

- $0 USD.

- Watermark.
- 5 exports/week.

## Server FFmpeg

- Server-side MP4 rendering.

## Unlimited Creator

- $7 USD/month.
- Unlimited no-watermark browser Record exports.
- Unlimited no-watermark server FFmpeg exports.

The no-server local app remains the baseline until the account/server layer is released.
