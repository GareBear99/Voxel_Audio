# Vault Edit and Re-export

Session Export Vault items can be edited and re-exported.

## Editable fields

Each completed browser Record item includes editable fields:

- Song name
- Artist
- Feat
- Description
- Export color

## Re-export

Click **Re-export** to:

1. Restore the vault item's metadata into the export panel.
2. Re-select the original track if it is still loaded.
3. Re-run the same Record mode, 10s or full.
4. Save the new output back into the Session Export Vault.

## Requirements

Re-export requires the original track and image binding to still exist in the current page session.

If the page was reloaded, re-load the folder and re-bind artwork first.
