# Session Export Vault Visibility

Browser Record outputs appear in the Session Export Vault.

## Where it is

Use the fixed Quick Dock button:

```text
Vault
```

The vault also opens automatically after a successful browser Record export.

## What appears there

The vault stores the last 9 completed browser Record outputs in memory:

- 10s Record exports
- Record Full exports

Each item has a Download button.

## Important

The vault is session-only. It clears when the page is closed or reloaded.

The vault does not store FFmpeg Kit downloads or Album Kit downloads.
