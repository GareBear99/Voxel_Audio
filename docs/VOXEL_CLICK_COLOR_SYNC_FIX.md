# Voxel Click Color Sync Fix

Clicking/selecting a voxel now resets the export color back to that selected voxel color.

## Behavior

- Select/click a voxel: export color syncs to that voxel.
- Use Export color picker: manual override applies.
- Select/click another voxel: manual override clears and the new voxel color becomes active.

## Crosscheck

Preflight shows whether the active color is:

- selected voxel
- manual override
