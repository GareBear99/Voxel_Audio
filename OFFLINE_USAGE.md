# Offline / No-Server Usage

Voxel Audio can run offline as a local file.

## Steps

1. Download or clone the repo.
2. Open `index.html`.
3. Load an audio folder.
4. Bind artwork.
5. Preview or record.

## Final MP4 output

For final MP4 output, install FFmpeg and use **Build MP4 Kit**.

The generated FFmpeg scripts also run offline.
